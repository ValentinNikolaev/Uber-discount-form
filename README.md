AfterUbercart form
============
This is form for Uber taxi to get estimate price for a trip with some discount. Discount can be set from configuration.

What's inside:
-------------
Slim, 
Twig (inc. slim twig-view), 
Monolog,
Bootstrap, 
Google Maps

Installation:
-------------
* composer install
* chmod 777 -R log/
* chmod 777 -R cache/
